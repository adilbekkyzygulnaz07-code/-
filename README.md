# Movie Ticket Booking System

## О проекте  
Это простая консольная система для бронирования билетов в кино (кинотеатр).  
Пользователь может выбрать зал, фильм, сеанс, зарезервировать одно или несколько мест, подтвердить бронь (имитация оплаты), отменить её — и всё это с учётом логики освобождения/продажи мест.  
Проект написан на Python с применением ООП и минимальными зависимостями (только стандартные библиотеки).

## Функциональность  
- Создание зала (с количеством рядов и мест)  
- Управление расписанием сеансов (добавление фильмов + времени)  
- Просмотр и управление статусом мест: свободно / зарезервировано / продано  
- Бронирование мест  
- Подтверждение брони: проверка, что места зарезервированы + попытка оплаты через простую функцию `process_payment`  
- Отмена брони и освобождение мест  
- Простая демонстрация работы (файл `main_demo.py`)

Ключевая Функциональность и Структура:

Проект разделен на логические модули для лучшей поддерживаемости:

Файл / Папка

Определяет класс данных: Movie (Фильм)

theater.py

Определяет класс Theater (Сеанс), который управляет планом мест (матрицей мест) и их статусами (free, reserved, sold).

payments.py

Имитация платежа. Содержит функцию process_payment(amount).

booking.py

Содержит бизнес-логику для создания, резервирования мест, подтверждения (через оплату) и отмены бронирования. Реализует защиту от двойного бронирования.

main_demo.py

Точка входа и консольный демо-сценарий.

Статусы Бронирования

Система оперирует следующими статусами бронирования, определенными в требованиях:

pending (создана, но не оплачена)

confirmed (оплачена, место продано)

cancelled (отменена, места освобождены)

expired (опциональный статус для просроченных броней)
